P3bble

P3bble is an OpenSource C# .NET Framework for Pebble SmartWatch which runs on Windows Phone 8 and WinRT (Windows 8)

Currently the library is not finished. You can find the GitHub via https://github.com/p3root/P3bble

Features

    • Sending SMS Notifications to pubble (ToDo, will not know if Microsoft will allow to receive SMS notification in WP8. I still hope it to come with the next sdk)
    • Updating song metadata displayed on Pebble (Todo)
    • Retrieving and setting the time stored on Pebble (Todo)
    • Sending system messages to Pebble (Todo)
    • Pinging Pebble (Todo)
    • Restart Pebble (Done)


As said, the development stage is in an very young stage.
