Flint

Flint is an open-source C#/.NET implementation of pebble's bluetooth communication protocol, similar to Libpebble. It can be downloaded at https://github.com/barometz/flint.

Features

  • Sending email and SMS notifications to Pebble
  • Updating song metadata displayed on Pebble
  • Retrieving and setting the time stored on Pebble
  • Sending system messages to Pebble
  • Pinging Pebble

Flint does not contain any functionality to send or receive apps or firmware, although this is in the works.

Flintlock

Flintlock is a Windows desktop application that uses Flint. It doesn't do anything useful except for media/ppt control yet, but more is coming soonish.

http://barometz.github.io/flintlock/
