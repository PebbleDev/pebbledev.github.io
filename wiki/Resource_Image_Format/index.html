Resource Image Format

The .pbpack files contain multiple types of resources. The structure for the image ("png", "png-trans") files is as follows:

Offset Description
0x0 Number of bytes used per scanline (4, 8, 12, 16, 20)
0x2 Unknown number (always 4096)
0x4 Unknown number (always 0 for PNG images; some png-trans images have 5 instead)
0x8 Image width
0xA Image height
0xC-end Image data, 1 bit per pixel. The remaining bits after every scanline are filled with zeros to reach the specific "Number of bytes used per scanline".
