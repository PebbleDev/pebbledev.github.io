Hiding Built-In Apps

It is possible, by hacking firmware, to hide certain built-in apps (e.g. watchfaces). It will not grant extra banks for installing user apps though.

There is an address table in firmware, which holds links to 11 built-in apps:

  1. [Sliding] Text Watch
  2. Classic Analog
  3. Simplicity
  4. Music
  5. Sports (UUID: 4DAB81A6-D2FC-458A-992C-7A1F3B96A970)
  6. Golf (UUID: CF1E816A-9DB8-4511-BBB8-F60C48CA8FAC)
  7. Alarms
  8. Watchfaces (which displays list of all installed watchfaces)
  9. Settings
  10. Getting Started
  11. Date & Time

Starting address of that table changes between firmware versions. Some known values (as offsets in tintin_fw.bin) are shown below.

Each value takes 4 bytes. One may reposition these values. Also it is possible to remove (disable) some items from that table. The simplest way to skip items from start of table (i.e. watchfaces) is to change byte in loader function from 0 to 4*(number of items to skip). Addresses of that byte are also shown in table below.

Version HW Table "Skip" byte
1.13.0 V2R2 0x36F68 0x6FE0
1.13.0 V3R1 0x370C4
1.14.1 V2R2 0x389E0 0x75B8
1.14.1 V3R1 0x38B40
2.0 BETA3 V2R2 0x3C424 0x1112A
2.0 BETA3 V3R1 0x3C588
