Hardware

  • STM32F205RE Cortex M3 CPU (Datasheet)
    • 1024KB Flash Memory
    • RAM between 64kb and 128KB
    • Pinout: Hardware/Pinout
  • Micron N25Q032A11ESE40F 32 Mb serial flash (Datasheet) (SPI1, Chipselect _maybe_ on GPIOA Pin 4)
  • PAN1316 Bluetooth Module (Datasheet) - (Connected to UART1)
    • Texas Instruments CC2560A with CC2564 firmware
  • Sharp LS013B7DH01 Display (Datasheet) (SPI2, Chipselect on GPIOB Pin 12)
  • STMicroelectronics LIS3DH Accelerometer (Datasheet) - (Connected to I2C1)
  • Fullriver LiPoly Battery (3.7V 130mAh) -- Battery Lifetime FAQ, or "Why doesn't pebble have a fuel gauge?"
  • Freescale Semiconductor Xtrinsic MAG3110 Magnetometer (Datasheet) (Connected to I2C2)
  • Unknown Ambient Light Sensor
  • UART3 - Debug Serial Console
  • Buttons:
    • All buttons are active-low (e.g. considered pressed when Pin is low)
    • Back button - GPIOC, Pin 3
    • Up button - GPIOA, Pin 2
    • Enter button - GPIOC, Pin 6
    • Down Button - GPIOA, Pin 1
